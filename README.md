# Dice Probability Calculator - Cresting Lilypads

A small Java tool for calculating dice outcome probabilities for the *Cresting Lilypads (Naruto AU)* quest on Spacebattles.

It estimates win and critical win chances based on the quest’s d10-based success system and allows comparing outcomes between different dice pools.

Quest: https://forums.spacebattles.com/threads/cresting-lilypads-naruto-au-quest.1270316/

---

## Features

- Calculates probability outcomes for dice rolls  
- Supports single and opposed roll comparisons  
- Can evaluate multiple enemy dice values or ranges  
- Outputs win and critical win chances
